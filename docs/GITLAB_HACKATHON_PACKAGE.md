# Gemini Live Agent Challenge Submission Package

FabPilot Live X is not a chatbot. It is an operator-grade semiconductor command surface for high-trust AI software workflows.

## Why it fits the hackathon
- clear multimodal mission flow
- reviewable evidence graph and decision trace
- approval-gated action instead of fake autonomy
- useful artifacts: brief, handoff, mission pack, replay context

## Demo arc
1. Trigger a risky semiconductor mission
2. Assemble evidence graph
3. Show ranked decision trace
4. Stage approval-gated action
5. Export signed handoff / debug artifact

## What judges should remember
- the system is bounded
- the system is inspectable
- the system is useful in a real workflow
- the UI feels like operational software, not AI theater
